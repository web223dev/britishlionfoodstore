== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use Product Carousel in your post, page, widget or anywhere you like. How to create Product Carousel? lets see.


1. Upload the folder product-carousel to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
3. Go to Product Carousel > Add New Shortcode> Publish.
4. Call shortcode to your WordPress editor directly.
